# AlcoLive — Bar Open Alpha Overhaul

A classic-modern creator link hub for AlcoLive. This overhaul replaces the CSS-art bottle experiment with generated transparent bottle assets, a cohesive bar-stage background, and a simpler link structure.

## Install

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Assets

## Notes

- No `dist/`, `node_modules/`, or `package-lock.json` are included.
- The bottles are temporary generated assets with transparent backgrounds.
- The neon sign is CSS text/effects so it can be animated and edited quickly.
